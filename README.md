# 🎵 Eksplorasi Fitur Audio Spotify

Proyek ini adalah visualisasi interaktif untuk mengeksplorasi hubungan antara berbagai fitur audio (danceability, energy, valence, tempo, dll.) dengan popularitas lagu dari dataset Spotify (114k lagu).

**Live Demo:**  
[https://USERNAME_GITHUB.github.io/REPO_NAME/](https://USERNAME_GITHUB.github.io/REPO_NAME/)  
*(Ganti dengan URL GitHub Pages kamu)*

---

## 📊 Fitur Interaksi

- **Pilih sumbu X dan Y** dari dropdown untuk mengeksplorasi berbagai pasangan atribut audio.
- **Filter genre** dengan centang pada 12 genre paling umum.
- **Tooltip detail** muncul saat hover di atas titik: menampilkan judul lagu, artis, genre, dan nilai atribut.
- **Animasi transisi** halus saat data berubah (durasi 750ms).

## 🧪 Cara Menjalankan

1. Buka `index.html` di browser.
2. Klik area upload atau *drag & drop* file `dataset.csv` (sudah tersedia di repositori ini).
3. Gunakan kontrol untuk mulai eksplorasi data.

## 📁 Sumber Data

[Spotify Tracks Dataset (Hugging Face)](https://huggingface.co/datasets/maharshipandya/spotify-tracks-dataset)  
Diunggah sebagai `dataset.csv` dalam repositori ini.

## 🖥️ Teknologi

- **D3.js v7** untuk visualisasi dan interaksi.
- **HTML5 + CSS3** untuk antarmuka.
